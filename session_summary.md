# Z.ai grid tool — session

A single-file browser tool that traces the Z.ai logo and overlays an editable grid locked to the logo's two stroke angles (plus an adjustable-angle third guide family), with click-to-fill parallelogram cells. The tool supports pan/zoom, undo, right-click delete, shift-select, SVG import/export, and live-reshaping cells bound to their guides — for constructing matching "AI" letterforms beside the Z.
